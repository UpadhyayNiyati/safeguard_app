🚀 QUICK START (5 MINUTES)
================================================================================

STEP 1: Create Virtual Environment
   $ python -m venv venv
   
   On Windows:
   $ venv\Scripts\activate
   
   On macOS/Linux:
   $ source venv/bin/activate

STEP 2: Install Dependencies
   $ pip install -r requirements.txt

STEP 3: Create .env File
   $ cp .env.example .env
   
   Edit .env and add:
   - FLASK_SECRET_KEY=generate_random_key_here
   - MAIL_USERNAME=your_gmail@gmail.com
   - MAIL_PASSWORD=your_gmail_app_password

   ⚠️ See "EMAIL SETUP" below for Gmail app password

STEP 4: Run the Application
   $ python app.py

STEP 5: Open in Browser
   Visit: http://localhost:5000

   🎉 YOU'RE DONE! The app is running!

📧 EMAIL SETUP (GMAIL)
================================================================================

To enable OTP email verification:

1. Go to: https://myaccount.google.com/
2. Click "Security" on the left side
3. Enable "2-Step Verification" (if not already enabled)
4. Search for "App Password" in security settings
5. Select "Mail" and "Windows Computer"
6. Google will give you a 16-character password
7. Copy that password into .env as MAIL_PASSWORD

⚠️ DO NOT use your regular Gmail password!
   Use the 16-character app password from step 6

🚀 DEPLOYMENT OPTIONS
================================================================================

HEROKU:
  1. Install Heroku CLI
  2. Add Procfile: echo "web: gunicorn app:app" > Procfile
  3. Run: git push heroku main

AWS / DIGITALOCEAN:
  1. Install Python 3.8+
  2. Clone repository
  3. Set environment variables
  4. Run: gunicorn -w 4 app:app

DOCKER:
  1. Create Dockerfile (template provided in docs)
  2. Build: docker build -t safety-assistant .
  3. Run: docker run -p 5000:5000 safety-assistant

LOCAL PRODUCTION:
  1. Set FLASK_SECRET_KEY to random value
  2. Use production WSGI server (gunicorn)
  3. Enable HTTPS
  4. Set up logging
  5. Regular backups

See SETUP_GUIDE.md for detailed deployment instructions

🐛 TROUBLESHOOTING
================================================================================

ERROR: "ModuleNotFoundError: No module named 'flask'"
  ✅ Solution: pip install -r requirements.txt

ERROR: "Address already in use"
  ✅ Solution: Change port in app.py or kill process using port 5000

ERROR: "Email not sending"
  ✅ Solution: Check .env has correct MAIL_USERNAME and MAIL_PASSWORD
  ✅ Solution: Verify Gmail 2-Step Verification is enabled
  ✅ Solution: Use Gmail app password, not regular password

ERROR: "Location not working"
  ✅ Solution: Allow browser location permission
  ✅ Solution: Check internet connection
  ✅ Solution: Try incognito/private mode

ERROR: "Database locked"
  ✅ Solution: Delete safety_app.db and restart app
  ✅ Solution: Database will be recreated automatically

For more help, see SETUP_GUIDE.md "Troubleshooting" section
