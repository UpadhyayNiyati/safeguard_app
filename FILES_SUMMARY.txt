================================================================================
QUICK START:
================================================================================

1. python -m venv venv
2. source venv/bin/activate (or venv\Scripts\activate on Windows)
3. pip install -r requirements.txt
4. Create .env file (copy .env.example and add your Gmail app password)
5. python app.py
6. Visit http://localhost:5000

✨ That's it! Your Safety Assistant is ready to use!

IMPORTANT NOTES:
================================================================================

⚠️  EMERGENCY DISCLAIMER:
    This app is a supplementary tool. Always call 911 first in emergencies!

🔒 SECURITY:
    - Keep .env file private (never commit to git)
    - Use strong FLASK_SECRET_KEY (not "supersecretkey")
    - Use Gmail App Password (not regular password)
    - Enable 2-Step Verification on Gmail

📧 EMAIL SETUP:
    - Use Gmail with App Passwords (created in Google Account)
    - Do NOT use regular Gmail password
    - 16-character app password from Google

🌐 DEPLOYMENT:
    - Change FLASK_SECRET_KEY for production
    - Use environment variables for all secrets
    - Enable HTTPS
    - Use production WSGI server (gunicorn)
    - Set up logging and monitoring

📱 MOBILE:
    - App is fully responsive
    - Works on iOS and Android
    - Location access may need mobile browser permission
