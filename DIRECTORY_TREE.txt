================================================================================
DATABASE STRUCTURE
================================================================================

USER TABLE:
├── id (Integer, PK)
├── email (String, Unique)
├── is_verified (Boolean)
├── last_otp_sent (DateTime)
└── contacts (Relationship → EmergencyContact)

EMERGENCYCONTACT TABLE:
├── id (Integer, PK)
├── name (String)
├── phone (String)
├── email (String)
├── relationship (String)
├── user_id (FK → User)
└── created_at (DateTime)

SAFETYLOG TABLE:
├── id (Integer, PK)
├── user_id (FK → User)
├── timestamp (DateTime)
├── lat (Float)
├── lon (Float)
├── status (String)
└── amenity_type (String)
